# 📄 Personal CV Website

This project is a **Personal CV (Curriculum Vitae) Website** built using **HTML** and **CSS**, designed to present my academic background, technical skills, and projects in a clean and structured format.

## 👨‍💻 About the Project

The purpose of this project is to create a digital version of my CV that can be easily shared online. It highlights my profile as a **Software Engineering student** and showcases my development skills.

## 🚀 Features

* Clean and professional layout
* Structured CV sections
* Responsive table design for education and languages
* Highlighted skills and projects
* Easy-to-read typography and spacing

## 🧾 Sections Included

* 📌 Personal Information
* 🙋‍♂️ About Me
* 🎓 Education & Training
* 💻 Technical Skills
* 📂 Projects
* 🤝 Soft Skills
* 🌐 Language Skills
* 📢 Conferences & Seminars
* 📜 Certifications
* 📬 Recommendation

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📂 Projects Highlighted

* Student Management System (Python)
* Sales & Revenue Tracker (C#)
